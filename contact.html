Yes! We're students ourselves and believe habit-tracking tools should be accessible to everyone. We have no plans to charge for basic features, though we might offer optional premium features in the future to help cover server costs.

We love feature suggestions! The best way is to tweet us @HabitHeroApp or open an issue on our GitHub repository. We prioritize features that get the most student requests.

Currently we focus on individual habit tracking, but we're working on study group features that will let you track shared habits with classmates. Expect this in our next big update!

We welcome contributors! Check our GitHub for open issues labeled "good first issue" if you're new to open source. We also need help with documentation, design, and user research.
